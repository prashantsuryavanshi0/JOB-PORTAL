## Project Overview

The **Job Portal System** is a full-stack web application built using the **MERN stack** that connects **Employers**, **Candidates**, and **Admins** on a centralized recruitment platform.

The system allows employers to post jobs, candidates to apply for jobs with resume upload, and admins to verify employers and manage job listings.

---

## Objective

To develop a **backend-driven Job Portal System** that enables:

- Job postings by employers
- Job applications by candidates
- Administrative verification and monitoring
- Secure, role-based authentication

---

## Tech Stack

### Frontend

- **React (Vite)**
- **Tailwind CSS**
- **Redux Toolkit** (state management)
- Axios

### Backend

- **Node.js**
- **Express.js**
- **MongoDB** (Mongoose)

### Tools

- **Postman** (API testing)
- Git & GitHub

---

## System Modules

### Employer Module

- Employer registration & login
- Post new job openings
- View applicants for posted jobs

### Candidate Module

- Candidate registration & login
- View available jobs
- Apply for jobs
- Upload resume

### Admin Module

- Verify employers and job postings
- Manage users and job listings
- Monitor application flow

---

## Functional Requirements

- Role-based authentication (Admin / Employer / Candidate)
- Job posting and application workflow
- Resume upload functionality
- Admin verification system
- Secure database transactions
- RESTful API architecture
